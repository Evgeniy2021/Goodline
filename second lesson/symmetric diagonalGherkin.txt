Функции(Feature): Создать двумерный массив, определить и вывести, 
                  симметрична ли матрица относительно главной диагонали
Сценарий: Создание двумерного массива
     Дано не задан размер массива
     То программа просит указать размерность массива <"Укажите размерность массива: ">
     И вводим число <4>
     И программа должна создать двумерный  массив размером(вместимосью) <[4,4]> 
  
Сценарий:Заполнение массива числами вручную с клавиатуры
     Дано ввели размер массива
     То должно появиться сообщение-предложение ввода элементов значений массива <"Введите элементы массива"> 
     Когда ввели <16> значений 
     То программа начнёт сравнивать значения по диагонали

Сценарий:Определение симметричности массива
     Дано ввели размер массива
     И ввели значения массива
     То программа в цикле сравнивает число одной строки с числом другой строки по диагонали
     Когда значения элементов не равны между собой
     То переменной <symmetric> присваивается выражение <false>
     
Сценарий:Вывод сообщения о симметричности диагонали массива
     Дано ввели правильно размер массива
     И введены правильно значения массива
     Когда переменная <symmetric = false>
     То программа ввыведет сообщение <"Диагональ массива не симметрична!">
     Когда переменная <symmetric = true>
     То программа ввыведет сообщение <"Диагональ массива симметрична!">
Примеры:
         |Размер массива|  Элементы массива |   Сообщение о симматричности    | 
         |   (size)     |  (array[i, j] )   |         (symmetric)             |
         |--------------|-------------------|---------------------------------|
         |              |   2   1   1       |                                 |
         |              |         *         |           "Диагональ            |
         |     3*3      |   5   1   10      |             массива             |
         |              |     *             |           симметрична!"         |
